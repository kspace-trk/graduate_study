\chapter{評価実験}

本研究の有用性を検証するため，20代3名のプレイ情報を教師データとして各多数決機械の荷重ベクトルと基準値を前もって学習させ，教師データを提供した3名に加えて20代の10名を被験者として評価実験を行った．

\section{方法}

被験者はポーカーシステムにおける以下の各対戦を順番にプレイし，ゲーム数が30回に到達する，もしくは，被験者かCOMのどちらかが持ち金10コイン以下となった時点でゲームを終了とする．各対戦後，アンケートに回答し，全対戦後に実験を通じた総合的なアンケートに回答する．1と2の対戦に関しては表情情報を考慮しないため，インタフェース上にユーザおよびCOMの表情を出力しない．

\begin{enumerate}
	\item 確率に基づく表情を使用しないCOMとの対戦
	\item 多数決機械を用いた表情を使用しない COMとの対戦
	\item 多数決機械を用いた表情を使用するCOMとの対戦
\end{enumerate}

また，評価実験を行うにあたり，テキサス・ホールデムの説明も兼ねて，サイトにて被験者に実験の説明を行った．付録Aに使用したサイトを掲載する．

\section{アンケート}
被験者に対して複数の視点からなるアンケートを実施した．
対戦後に行ったアンケートの設問内容は以下のとおりである．

\begin{itemize}
	\item 基本情報
	      \begin{itemize}
		      \item 年齢
		      \item 性別
		      \item テキサス・ホールデムのプレイ経験（4段階）
	      \end{itemize}
	\item 最終結果
	      \begin{itemize}
		      \item ユーザの最終持ち金
		      \item COMの最終持ち金
		      \item ゲームの継続数
	      \end{itemize}
	\item COMからユーザへのブラフ
	      \begin{itemize}
		      \item COMがユーザにブラフをかけていると感じたか（5段階）
		      \item 感じた場合の評価（5段階）と理由（自由記述）
		      \item 感じなかった場合の理由（自由記述）
	      \end{itemize}
	\item ユーザからCOMへのブラフ
	      \begin{itemize}
		      \item ユーザからCOMへブラフを仕掛けたか （3段階）
		      \item 仕掛けた場合の評価（5段階）
		      \item 仕掛けなかった場合の理由（自由記述）
	      \end{itemize}
	\item COMの行動パターン性
	      \begin{itemize}
		      \item COMの行動にパターン性を感じたか（4段階）
		      \item パターン性を感じた場合の理由（自由記述）と感じたことによるゲームへの印象（4段階）
		      \item パターン性を感じなかった場合の理由（自由記述）と感じなかったことによるゲームへの印象（4段階）
		      \item COMがユーザの行動に合わせて行動を変えていると感じたか（5段階に加えて，覚えていない）
	      \end{itemize}
\end{itemize}

また，全対戦後に実施したアンケート内容は以下のとおりである．

\begin{itemize}
	\item インタフェースの使いやすさ（5段階）
	\item COMの表情がゲームを面白くすることに影響を与えていたか（4段階）
	\item 研究に関する意見（自由記述）
\end{itemize}

対戦1後に実施したアンケートを付録Bに，すべての対戦後に使用したアンケートを付録Cに掲載する．対戦2,3後のアンケートは付録Bと同様である．

\section{結果}
