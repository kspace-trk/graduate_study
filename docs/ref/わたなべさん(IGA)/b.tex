\chapter{被験者の自由記述}

アンケートの各自由記述欄に記入された意見を以下に記す．

【インタフェースについて感じたこと】
\begin{itemize}
\item 特にありません．
\item guiに手順が使い方が書いてあったほうがいいと思います
\item 曲が新しくなったことがわかりづらい
\item インターフェースは見やすかった
\end{itemize}


【飽きた理由】
\begin{itemize}
\item 曲の変化がつかみにくかった．
\item 似た曲を繰り返し聞く
\item 差が少ない気はする
\item 楽器やコード進行の変化に乏しく，フレーズの以外の変化を感じ取ることが難しい
\item 変化がややすくない
\item 6曲の違いが小さかったため選ぶのに悩んでしまい時間が掛かった
\end{itemize}

【全体を通しての感想，意見】
\begin{itemize}
\item 好みの音に近づいていく感じがとても面白かったです．最後の方にはどれも好みに近いものになっていたためとてもよいものだと思います．
\item やってることが面白いと思います！これまで音楽を作るのは人が主体だったのに対し，人が機械に代わり，テクノロジーの発展を実感しました．
\item 思っていたよりも好みに近づいた
\item 純粋に楽しかった．気に入った曲調が選択肢に増えていくのが面白かったです．
\item さらに素材が増えて，曲調の幅が広がっても同じくらいの早さで作曲出来るようになったら面白そうだなと思いました．
\item 曲同士の違いがよくわからない時が．
\item 機能がシンプルで良い
\item 最初からそれなりに気に入ったのであまり変化を感じなかった
\item IGAの自動作曲全般で，フレーズの区切りが予想外のところに来ることが多いので，フレーズの意識をもっと持たせられないかと思います
\item 曲調のバリエーションがわかりづらい
\item 最初から最後までの曲の変化が小さく感じた
\item しかし個人的には好きな感じの曲ができ満足
\item 卒研頑張ってください！
\end{itemize}
